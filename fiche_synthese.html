Orbi7rack

Suivi de colis en temps réel

Auteur : Thomas Virassamy
Titre : Développeur Web et Web Mobile
Session : Juillet 2026
Description

Orbi7rack est une application full-stack de suivi de colis qui remplace le suivi textuel classique par un globe 3D interactif (globe.gl + Three.js). Chaque colis est représenté par un arc lumineux animé entre son origine et sa destination, avec un marqueur pulsant indiquant sa position courante.

Quand un numéro de vol est identifié dans les événements 17TRACK, la position de l'avion cargo est récupérée en direct via OpenSky Network. En l'absence de données live, un SimulationEngine calcule la progression interpolée (slerp Haversine multi-segments) à partir des événements géocodés pour garantir une visualisation continue.

Pipeline de données — du numéro de tracking au globe
01
Ajout du colis & polling Celery
L'utilisateur saisit un numéro de tracking → tâche Celery périodique → appel API 17TRACK → persistance Parcel + TrackingEvent
tasks.py · models.py
02
Extraction du vol & identification cargo
Regex [A-Z]{2,3}\d{3,4} sur les descriptions · mapping carrier→IATA (UPS→5X, FedEx→FX…) · fallback AviationStack
flight_extractor.py
03
Géocodage & chaîne de fallback
1. Position live OpenSky · 2. geopy sur l'événement · 3. centroïde pays destination · 4. centroïde pays origine
geocoding.py
04
SimulationEngine — mode & progression
Détection air/road/sea · Haversine multi-segments · progress(t) → slerp · flag simulated: bool
simulation_engine.py
05
Rendu Globe 3D
Hexagones H3 · atmosphère bloom (UnrealBloomPass) · arcs cyan animés · marqueurs pulsants · sprites avion/camion orientables · zoom caméra 1,2 s
Globe.tsx · useFlightPosition.ts
